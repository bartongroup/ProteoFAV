ProteFAV is a Python framework to fetch, process and integrate protein structure and features
to genetic variants. The tool relies heavily in `Pandas`_ library.

ProteFAV benefits
~~~~~~~~~~~~~~~~~

- Fast
- Scalable
- Ready
